# java-chess 게임

- [x] 체스판
    - [x]rank(세로) : 8 :1~8
    - [x]file(가로) : 8 :a~h
    - [x] 말이 없는 칸은 .
  - [x]초기화

- [] 아웃풋 뷰
    - [] 체스판 출력 기능
      - [] 게임 진행시 마다 체스판을 출력해준다
  - [x] 게임 시작 설명 출력
    
-[] 인풋 뷰
  - [x] 게임 명령어 인풋 받기
    - [x] start/end/move로 시작하지 않으면 계속 묻기
    
  
- [] 기물 
  - [] king
     - [] 상하좌우 중 1칸 이동  
  - [] queen
    - [] 상하좌우 대각선 n칸 이동
  - [] bishop
     - [] 대각선으로 n칸 이동
  - [] knight
    - [] 사방중 한방향으로 한칸 & 양 대각선 중 한 방향으로 한칸 으로 이동
    - [] 상대 말 넘어감
  - [] rook
      - [] 상하좌우 n칸 이동
  - [] pawn
    - [] 첫회 한정: 2칸 상 이동 || 1칸 상 이동
    - [] 1칸 이동
    - [] 공격은 대각선으로만
  
- [x] 진영
  - [x] 기물 2 세트 생성 (1개는 대문자, 소문자)
  
- [] 체스 게임 룰
  - [] start : 체스판 초기화 및 진영 세팅
  - [] end: 게임 종료
